### 2번 시도: 해결

{% highlight python %}
from datetime import datetime
import pytz

class TimeTracker:
    start_time = ""
    stop_time = ""

    def start(self):
    	self.start_time = datetime.now(pytz.timezone('Asia/Seoul'))
    	return self.start_time

    def stop(self):
    	self.stop_time = datetime.now(pytz.timezone('Asia/Seoul'))
    	return self.stop_time

    def get_elapsed_time(self):
    	return (self.stop_time - self.start_time).total_seconds() / 60

study_session = TimeTracker()
study_session.start()
study_session.stop()
study_session.get_elapsed_time()
{% endhighlight %}

**Feedback:** 사용자가 실수로 `stop()`을 실행시키지 않았을 경우를 생각하지 않음
