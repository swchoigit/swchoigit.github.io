### 결측치 확인
`company`의 결측치가 가장 높지만 값이 없다는 것은 개인이 예약을 했다는 것이므로 값이 없다고 행을 지워서는 안 된다. 대신, 값이 없는 것은 0으로 채운다.
{% highlight python %}
df.isnull().sum().loc[lambda x : x > 0].sort_values(ascending=False)
df = df.fillna(0)
{% endhighlight %}

### 예약 취소율
예약을 취소한 사람들뿐만 아니라 노쇼도 매출 저하에 영향을 주기 때문에 두 가지 상태 모두 생각해야 한다.
{% highlight python %}
print(f"{df['reservation_status'].isin(['Canceled', 'No-Show']).mean():0.1%}")

## 취소 원인 분석

### 이전 방문 여부에 따른 분석
`is_repeated_guest = 0`이면 신규 고객이고, `is_repeated_guest = 1`이면 기존 고객이다. 취소 고객의 대부분이 신규 고객이었다. 왜 신규 고객들이 예약을 취소하게 되었는지 알아보자.

{% highlight python %}
first_time_guest = (df['reservation_status'].isin(['Canceled', 'No-Show'])) & (df['is_repeated_guest'] == 0)
print(f"{first_time_guest.mean():0.1%}")
{% endhighlight %}

### 예약일에 따른 분석
`lead_time`이 0~300 사이인 예약들, 즉 예약 당일부터 300일 전까지의 예약들의 취소율이 가장 높았다.

{% highlight python %}
plt.figure(figsize=(10, 5))

## 분석 결과
### 1. 신규 고객 유치를 위한 방안
예약 취소 인원 대부분이 신규 고객이다. 예약을 했더라도 서비스 품질이나 리뷰가 좋지 않은 경우, 혹은 광고된 내용과 실제 내용이 다른 경우 고객이 예약을 취소할 수도 있다. 신규 고객 유치를 위한 방안이 필요한 것으로 보인다.

### 2. 취소와 노쇼에 대한 패널티 제도 강화
예약 당일에 취소하거나 노쇼 하는 사람들에 대한 패널티 제도가 부족했기 때문에 자유롭게 예약을 취소할 수 있었던 것으로 보인다.


### 3. 불가피한 취소에 대한 대안 마련
환불이 불가한 경우에도 취소를 한 사람들에 대해서 원인을 분석하고, '예약 취소' 대신 '예약일 변경' 등의 대안을 생각하면 좋을 것 같다.
