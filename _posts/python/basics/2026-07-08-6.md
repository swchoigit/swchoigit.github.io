## 1차 시도: 실패

```python
def is_prime(num):
  prime_list = [1, 2, 3, 5, 7]
  sum = 0

  if num in prime_list or (num not in prime_list and (num % 2 != 0 or num % 3 != 0)):
      return True

  print(sum)
```

**Feedback:**
- `if` 문의 위치
- `i`의 사용 부재

## 2차 시도: 실패

```python
def is_prime(num):
  if num < 2:
    return False
  elif num == 2 or (num-1) % 2 == 0:
    return True
```

## 3차 시도: 실패

```python
def is_prime(num):
  if num < 2:
    return False

  for i in range(2, num):
    if (num % i == 0):
      return True
    else:
      return False
  
  return True
```

## 4차 시도: 해결

```python
def is_prime(num):
  if num < 2:
    return False

  for i in range(2, num):
    if (num % i == 0):
      return False

  return True
```
