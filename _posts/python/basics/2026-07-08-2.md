## 1차 시도: 실패

```python
class ReservationSystem:
	restaurants = {}
	branch_name = ""
	reservations = []

	def __init__(self, branch_name):
		self.branch_name = branch_name
		ReservationSystem.restaurants[branch_name] = []
	
	def add_reservation(self, customer, date, count):
		self.reservations.append([customer, date, count])
		ReservationSystem.restaurants[self.branch_name] = [customer, date, count]
```

**Feedback:** 인스턴스 변수를 클래스 변수 위치에 선언하지 않아도 된다.

## 2차 시도: 실패

```python
class ReservationSystem:
	restaurants = {}

	def __init__(self, branch_name):
		self.branch_name = branch_name
		ReservationSystem.restaurants[branch_name] = []

	def add_reservation(self, customer, date, count):
		self.reservations = []
		self.reservations.append([customer, date, count])
		ReservationSystem.restaurants[self.branch_name] = [customer, date, count]
```

**Feedback:**
- `self.reservations` 초기화 위치
- `self.reservations`를 활용하지 않음

## 3차 시도: 해결

```python
class ReservationSystem:
	restaurants = {}

	def __init__(self, branch_name):
		self.branch_name = branch_name
		self.reservations = []
		ReservationSystem.restaurants[branch_name] = []

	def add_reservation(self, customer, date, count):
		self.reservations.append([customer, date, count])
		ReservationSystem.restaurants[self.branch_name] = self.reservations
```
