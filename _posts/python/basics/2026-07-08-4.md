## 요구 사항:
- 직원 정보는 클래스 변수 `employees`에 저장되어 전체 `EmployeeManager` 인스턴스에서 접근 가능해야 합니다.
- `calculate_average_salary` 메서드는 저장된 모든 직원의 급여를 합산하여 평균을 출력하고, 직원이 없는 경우 0을 반환해야 합니다.

## 1차 시도: 실패

{% highlight python %}
class EmployeeManager:
	name = ""
	salary = 0
	employees = {}

def __init__(self, name, salary):
	self.name = name
	self.salary = salary
	self.employees[name] = salary
	
	print(f"{name} 님이 추가되었습니다. 급여: {salary}원")

def calculate_average_salary(self):
	avg_salary = 0

	if len(self.employees) == 0:
		return 0

	for value in self.employees.values():
		avg_salary += value
	
	avg_salary = int(avg_salary / len(self.employees))

print(f"전체 직원의 평균 급여: {avg_salary}원")
{% endhighlight %}

**Feedback:**
- `EmployeeManager`는 직원 개인이 아니라 직원 전체를 관리하는 시스템
- 불필요한 변수 선언: `name`, `salary`

**질문:** `__init__`에서 `self`가 가능한데 굳이 `EmployeeManager.employees`를 해야 할까?

## 2차 시도: 해결

{% highlight python %}
class EmployeeManager:
  employees = {}

  def __init__(self, name, salary):
    EmployeeManager.employees[name] = salary
    print(f"{name} 님이 추가되었습니다. 급여: {salary}원")

  @classmethod
  def calculate_average_salary(cls):
    avg_salary = 0

    if len(EmployeeManager.employees) > 0:
      for value in cls.employees.values():
        avg_salary += value

      avg_salary = int(avg_salary / len(cls.employees))

    print(f"전체 직원의 평균 급여: {avg_salary}원")
{% endhighlight %}
