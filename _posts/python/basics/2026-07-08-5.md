### Original Code

```python
import csv

def read_file(filename):
    f = open(filename, 'r')
    lines = f.readlines()
    f.close()

    result = []

    lines.pop(0)

    for l in lines:
        l = l.strip()
        result.append(l.split(","))

    return result
```

#### Feedback

| Before | After | Feedback |
| :--- | :--- | :--- |
| `f = open(...)`<br>`f.close()` | `with open(...)` | 예외가 발생해도 파일이 자동으로 닫혀 안전 |
| `lines.pop(0)` | &bull; `f.readlines()[1:]`<br>&bull; `next(f)` | &bull; `pop(0)`의 데이터 대이동 연산을 제거<br>&bull; `next(f)` 적용 시, 메모리를 통째로 쓰지 않는 이터레이터 방식이 되어 성능이 극대화 |
| `open(filename, 'r')` | `encoding='utf-8'` | 한글 깨짐 에러 방지 |

### Refactored Code

```python
def read_file(filename):
    with open(filename, 'r', encoding='utf-8') as f:
        next(f)
        return [line.strip().split(",") for line in f]
```
