## 1차 시도: 실패

```python
vowels = ["a", "e", "i", "o", "u"]

def remove_vowels(s):
    i = 0
    s_list = list(s)

    for i in range(i, len(s_list)):
        if s_list[i] in vowels:
            del s_list[i]

    return s_list
```

**Feedback:** 비교하려는 문자열의 길이가 맞지 않을 수도 있음

## 2차 시도: 실패

```python
vowels = ["a", "e", "i", "o", "u"]

def remove_vowels(s):
    s_list = list(s)

    for item in s_list:
        if item in vowels:
            s_list.remove(item)

    return s_list
```

**Feedback:** `remove()`는 첫 번째 요소만 제거함

## 3차 시도: 해결

```python
vowels = ["a", "e", "i", "o", "u"]

def remove_vowels(s):
	s_list = list(s)
	s_list = [x for x in s_list if x not in vowels]

    return s_list
```
