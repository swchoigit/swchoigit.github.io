## 취소 원인 분석

### 예약 취소율
예약을 취소한 사람들뿐만 아니라 노쇼도 매출 저하에 영향을 주기 때문에 두 가지 상태 모두 생각해야 한다. 전체 예약 중 37% 이상이 취소된 예약이다.

```python
canceled = df['reservation_status'].isin(['Canceled', 'No-Show'])
print(f"{canceled.mean():0.1%}")
```

#### 결과

```text
37.0%
```

### 취소율에 영향을 주는 수치형 변수들
아래 순서대로 취소율에 영향을 준다.

```python
correlation = df.corr(numeric_only=True)['is_canceled'].abs().sort_values(ascending = False).drop('is_canceled')
plt.figure(figsize=(10, 6))
sns.barplot(x=correlation.values, y=correlation.index, hue=correlation.index, palette='rocket', legend=False)
plt.title("취소율에 영향을 주는 수치형 변수들", fontsize=16, fontweight="bold")
plt.ylabel("")
plt.show()
```

### 남은 예약 일수에 따른 분석
예약 건수로 보면 예약 당일에 취소하는 사람이 압도적으로 많다. 취소율로 보자면 기간이 길어질수록 높아진다.

```python
fig, axes = plt.subplots(1, 2, figsize=(16, 6))

sns.kdeplot(data=df, x='lead_time', hue='is_canceled', fill=True, common_norm=False, alpha=0.5, ax=axes[0])
axes[0].set_title('남은 일수에 따른 취소 예약 수', fontsize=16, fontweight='bold')
axes[0].set_xlabel('Lead Time (Days)')

df['lead_time_month'] = (df['lead_time'] // 30) * 30
df_filtered = df[df['lead_time'] <= 360]

sns.lineplot(
    data=df_filtered, 
    x='lead_time_month', 
    y='is_canceled', 
    marker='o',
    color='#e74c3c',
    linewidth=2,
    errorbar=None,
    ax=axes[1]
)
axes[1].set_title('남은 일수에 따른 취소율', fontsize=16, fontweight='bold')
axes[1].set_xlabel('Lead Time (Days)')
axes[1].grid(True, linestyle='--', alpha=0.5)
```

### 특별 요청 수와 요청한 주차 공간 수에 따른 분석
특별 요청이 1개 이상부터 취소율이 급감하였고, 요청한 주차 공간 수가 1개 이상부터는 취소율이 0%였다.

```python
fig, axes = plt.subplots(1, 2, figsize=(16, 6))

# total_of_special_requests
sns.barplot(
    data=df,
    x='total_of_special_requests',
    y='is_canceled',
    errorbar=None,
    palette='Blues_r',
    ax=axes[0]
)

axes[0].set_title("특별 요청 개수에 따른 취소율 변화", fontsize=16, fontweight="bold")
axes[0].set_xlabel("Total Special Requests")

# required_car_parking_spaces
parking_ratio = pd.crosstab(
    df['required_car_parking_spaces'], 
    df['is_canceled'], 
    normalize='index'
) * 100

parking_ratio.plot(
    kind='bar', 
    stacked=True, 
    color=['#3498db', '#e74c3c'],
    rot=0,
    ax=axes[1]
)

axes[1].set_title("요청한 주차 공간 수에 따른 취소/유지 비율", fontsize=16, fontweight="bold")
axes[1].set_xlabel("Required Car Parking Spaces")
axes[1].legend(["유지", "취소 "], loc='lower left')

for p in axes[1].patches:
    width, height = p.get_width(), p.get_height()
    if height > 1.0:
        x, y = p.get_xy() 
        axes[1].text(
            x + width/2, 
            y + height/2, 
            f'{height:.1f}%', 
            ha='center', 
            va='center', 
            color='white', 
            fontweight='bold',
            fontsize=10
        )
```

## 분석 결과
### 1. 남은 일수별 고객 관리
* 예약 당일에 취소하는 사람들에 대한 패널티 제도 강화가 필요해 보인다.
* 특별 요청이나 요청한 주차 공간의 수가 많을수록 취소율이 현저히 적다. 예약일이 많이 남은 경우에는 사전에 특별 요청과 주차 공간 등에 대한 조사를 해서 고객에 예약에 대해 조금 더 관심을 둬서 변경을 쉽게 할 수 없게 만드는 방안이 필요해 보인다.

### 2. 신규 고객 유치를 위한 방안
예약 취소 인원 대부분이 신규 고객이다. 예약을 했더라도 서비스 품질이나 리뷰가 좋지 않은 경우, 혹은 광고된 내용과 실제 내용이 다른 경우 고객이 예약을 취소할 수도 있다. 신규 고객 유치를 위한 방안이 필요한 것으로 보인다. 

### 3. 불가피한 취소에 대한 대안 마련
환불이 불가한 경우에도 취소를 한 사람들에 대해서 원인을 분석하고, '예약 취소' 대신 '예약일 변경' 등의 대안을 생각하면 좋을 것 같다.
